# Financial Goal Calculator

Financial Goal Calculator is a client-side web application that estimates how long it will take to reach a savings goal based on user-defined inputs such as monthly savings, interest rate, and starting capital.

The application demonstrates financial calculation logic, compound interest modeling, and dynamic data visualization using JavaScript.

---

## Features

- Compound interest calculation
- Monthly savings simulation
- Time-to-goal estimation
- Real-time recalculation on input change
- Data visualization using charts
- Interactive UI with immediate feedback

---

## Tech Stack

- JavaScript
- HTML5
- CSS3
- Chart.js

---

## Core Logic

The calculation engine:

- Applies compound interest over time
- Simulates monthly contributions
- Updates balance iteratively until target amount is reached
- Dynamically updates chart visualization based on computed results

The project demonstrates structured client-side logic, data transformation, and UI synchronization.
